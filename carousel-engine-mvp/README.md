# Carousel Blocks Engine MVP

نسخة أولية تعمل مباشرة من المتصفح، بدون API.

## ما الذي تفعله؟
- تبني carousel على أساس **Blocks** بدل كتابة كل شريحة يدويًا.
- تعرض **Preview حي** بحجم 1080×1350.
- تسمح لك بتحرير:
  - المشروع والهوية
  - الشرائح
  - البلوكات
- تستورد وتصدر **JSON**.
- تحفظ تلقائيًا في `localStorage`.

## أنواع البلوكات المدعومة
- hero
- text
- checklist
- stat
- image-text
- cta

## طريقة التشغيل
افتح ملف `index.html` مباشرة في المتصفح.

## كيف أستخدمه مع ChatGPT؟
1. اطلب من ChatGPT أن يولد لك `Project JSON` بنفس البنية الموجودة في `sample-project.json`.
2. انسخ الناتج.
3. افتح تبويب JSON.
4. الصق المحتوى.
5. اضغط **استيراد JSON**.

## ملاحظات
- لا يوجد API ولا توليد آلي داخل التطبيق نفسه.
- هذه نسخة MVP، ومناسبة جدًا كقاعدة للبناء لاحقًا.
- إذا أردت لاحقًا:
  - drag and drop
  - export PNG
  - auto-fit للنص
  - templates متعددة
  - AI integration
  يمكن إضافتها فوق هذه البنية.
